# מה אוכלים השבוע 🍽️

אפליקציית תכנון ארוחות, בישולים וקניות למשפחה — עברית RTL, עם משתמש לכל בן בית וסנכרון חי בין המכשירים.
React (Vite) + Supabase (בסיס נתונים, התחברות, Realtime, ונעילת פרטיות ברמת-שורה).

## מה צריך
- Node 18+ במחשב.
- חשבון Supabase (יש חבילת חינם שמספיקה למשפחה) — https://supabase.com

## הקמה — 3 שלבים

### 1. בסיס הנתונים
- צור פרויקט חדש ב-Supabase.
- פתח **SQL Editor → New query**, הדבק את כל התוכן של `supabase/schema.sql`, ולחץ **Run**.
  (זה יוצר את הטבלאות, חוקי הפרטיות, וההצטרפות למשפחה.)
- ב-**Authentication → Providers → Email**: לשימוש משפחתי נוח אפשר לכבות "Confirm email" כדי שלא יידרש אימות מייל. (אופציונלי.)

### 2. חיבור המפתחות
- ב-Supabase: **Project Settings → API**. העתק את `Project URL` ואת `anon public key`.
- בתיקיית הפרויקט: העתק את `.env.example` ל-`.env` ומלא:
  ```
  VITE_SUPABASE_URL=...
  VITE_SUPABASE_ANON_KEY=...
  ```

### 3. הרצה
```bash
npm install
npm run dev      # מקומי, לבדיקה
```
לפריסה אמיתית (Vercel / Netlify):
- Build command: `npm run build`
- Output directory: `dist`
- הוסף את שני משתני הסביבה (`VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`) בהגדרות הפרויקט בענן.

## שימוש ראשון
1. אתה (הורה) נרשם ראשון → במסך הקליטה בחר **"משפחה חדשה"**. תקבל **קוד הצטרפות**.
2. כל שאר בני הבית נרשמים, בוחרים **"הצטרפות"** ומזינים את הקוד.
3. במסך **בנק המנות** לחץ **"טען מנות לדוגמה"** כדי להתחיל עם בסיס מנות.
4. במסך **התזונה שלי** עבור על העקרונות (טעונים מראש), ערוך אם צריך, ושמור. *רק אתה רואה את המסך הזה.*
5. שבץ ארוחות ב**לוח השבועי**, ואז ב**קניות** לחץ **"צור רשימה מהשבוע"**.

## מבנה
- `supabase/schema.sql` — כל בסיס הנתונים והפרטיות.
- `src/pages/` — המסכים (בית, לוח שבועי, בנק מנות, קניות, בישול, תזונה, משפחה, הדפסה).
- `src/data/` — מנות התחלתי + פרופיל התזונה (לעריכה חופשית).

הפרופיל התזונתי שלך נעול ברמת בסיס הנתונים (RLS) — אף בן משפחה אחר לא יכול לגשת אליו.
